# Enunciado

Realice una función `transponer()` que reciba como parámetro una matriz de enteros y que
devuelva la matriz transpuesta de la matriz recibida.

---

## Ejemplos

### Ejemplo 1
- **Input**: `{ {1, 2, 3}, {4, 5, 6}, {7, 8, 9} }`
- **Output**: `{ {1, 4, 7}, {2, 5, 8}, {3, 6, 9} }`

### Ejemplo 2
- **Input**: `{ {1, 2}, {3, 4}, {5, 6} }`
- **Output**: `{ {1, 3, 5}, {2, 4, 6} }`

### Ejemplo 3
- **Input**: `{ {1} }`
- **Output**: `{ {1} }`

### Ejemplo 4
- **Input**: `{ {1, 2, 3, 4} }`
- **Output**: `{ {1}, {2}, {3}, {4} }`
